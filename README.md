## Datasets setting 
The directory structure of the dataset is as follows:
```text
<dataset_root>/
├── test/                 # Test set directory (only input images, no GT)
│   └── [test image files]
├── train/                # Training set directory
│   ├── gt/               # Training ground truth images directory
│   │   └── [training GT images]
│   ├── input/            # Training input images directory
│   │   └── [training input images]
│   └── traindata_list.txt # Generated by extract_name.py
├── valid/                # Validation set directory
│   ├── gt/               # Validation ground truth images directory
│   │   └── [validation GT images]
│   ├── input/            # Validation input images directory
│   │   └── [validation input images]
│   └── validdata_list.txt # Generated by extract_name.py
├── traindata_list.txt     # Generated by extract_name.py
└── validdata_list.txt     # Generated by extract_name.py

### Train

An example of training on UIEB
```
 python train.py -d UIEB --cuda --b 32 --ckpt-save-path ../ckpts --nb-epochs 500  
```
### Test
An example of testing on UIEB
```
 python src/test_PSNR.py --dataset-name UIEB --path ./data/UIEB/test   
```
